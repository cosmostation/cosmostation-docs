# BalanceOf

## **[GET]** /v1/evm/:network/token/erc721/balance

`function balanceOf(address owner) view returns (uint256)`

Get ERC721 token Balance of account

_ERC165 interface id: `0x80ac58cd`_

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: kava
```

```yaml filename="Queries"
# The contract address of ERC721 token
# (string, required)
contract_address: 0x04866796aabB6B58e6bC4d91A2aE99105b2C58AE

# The address of account
# (string, required)
address: 0x8d3Debe408858a8Abe2EA977BDC18EC8201C05D7
```
