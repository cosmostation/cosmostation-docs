# BalanceOf

## **[GET]** /v1/evm/:network/token/erc1155/balance

`function balanceOf(address owner, uint256 id) view returns (uint256)`

Get ERC1155 token Balance of account

_ERC165 interface id: `0xd9b67a26`_

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: kava
```

```yaml filename="Queries"
# The contract address of ERC1155 token
# (string, required)
contract_address: 0x2e779749c40cc4ba1cab4c57ef84d90755cc017d

# The address of account
# (string, required)
address: 0xc6d1c9b8e605c0777ffd7a7528a5e8b98900ebe1

# The id of token
# (string, required)
id: 25
```
