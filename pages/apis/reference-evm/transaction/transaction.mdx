# Transaction by hash

## **[GET]** /v1/evm/:network/tx

Get a transaction by hash

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: kava
```

```yaml filename="Queries"
# The hash of transaction
# (string, required)
txhash: 0x680d039c939cbad982aedbf371d9bdefbaf99d7841486fec23f8ac14bd96e4dc
```
