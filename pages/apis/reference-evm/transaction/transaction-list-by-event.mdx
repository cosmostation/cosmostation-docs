# Transaction By Event (Topics)

## **[GET]** /v1/evm/:network/tx/list-by-event

Get Transactions by event

_events is filtered by address and topics_

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: kava
```

```yaml filename="Queries"
# required one of the following (address, topics)

# The address of topic
# (string, optional)
address: 0x52369b1539ea8f4e1eadeef18d85462dcf9a3658

# The topics of event
# (string, optional)
# topics can be separated by commas (,)
topics: 0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef

# The number of page to return
# (integer, optional, default: 1)
page: 1

# The number of transactions to return
# (integer, optional, default: 10)
offset: 10

# The order of transactions
# (string, optional, default: desc)
sort: desc

# Start searching for transactions
# (string, optional)
start_block: 1

# End searching for transactions
# (string, optional)
end_block: 99999999
```
