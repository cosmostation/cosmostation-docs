# Validator Voting Power Events

## **[GET]** /v1/:network/validators/:validatorAddress/powerEvents

Get voting power events of validator

### Request

```yaml filename="Headers"
Authorization: Bearer {access_token}
```

```yaml filename="Parameters"
# The name of network
# (string, required)
network: cosmos
# The address of account
# (string, required)
validatorAddress: cosmosvaloper1clpqr4nrk4khgkxj78fcwwh6dl3uw4epsluffn
```

```yaml filename="Queries"
# Reqeust per page
# (number, optional, default: 20)
take: 20
# Value of `pagination.searchAfter` from previous request for pagination
# (string, optional)
searchAfter: MTY4OTc1NTU4NjAwMA==
```
