## Grant

### Delegate Grant

```json
{
  "type_url": "/cosmos.authz.v1beta1.MsgGrant",
  "value": {
    "granter": "[input]",
    "grantee": "[input]",
    "grant": {
      "authorization": {
        "type_url": "/cosmos.staking.v1beta1.StakeAuthorization",
        "value": {
          "allow_list": {
            "address": ["[input]", "[input]"]
          },
          "authorization_type": 1 // [input]: number
        }
      },
      "expiration": {
        "type_url": "/google.protobuf.Timestamp",
        "value": {
          "seconds": 5325252355 // [input]: number
        }
      }
    }
  }
}
```
