# EVM

Cosmostation team provides an easy way to verify contracts.

## Kavascan.io

You can verify the contract on the respective contract page.

`https://kavascan.io/address/{contractAddress}?t=code`

## hardhat

Add customChain configuration. (`hardhat.config.ts`)

Although there is no speicifc API key distributed to the users, but the `apiKey` field is required to be entered.
Please enter a random value in the `apiKey` field

```
const config: HardhatUserConfig = {
  solidity: {
    // ...
  },
  networks: {
    hardhat: {},
    kava: {
      url: "https://rpc.kava.io",
      chainId: 2222,
      accounts: [],
    },
  },
  etherscan: {
    apiKey: {
      kava: "9MgwdOWmMs",
    },
    customChains: [
      {
        network: "kava",
        chainId: 2222,
        urls: {
          browserURL: "https://kavascan.io",
          apiURL: "https://api.verify.mintscan.io/evm/api/0x8ae",
        },
      },
    ],
  },
};
```
